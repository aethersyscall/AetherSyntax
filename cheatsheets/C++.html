C++

Zero-Cost Abstractions.

C++ allows you to write high-level code with low-level performance. It introduces Object-Oriented Programming (OOP) and Generic Programming to the systems world. It is the language of game engines, high-frequency trading, and browsers.

I. Novice (The Basics)

C++ adds a standard library and streams to the C foundation.

Hello World

Input/Output is handled by streams (iostream). std::cout prints to console.

cpp
#include <iostream>

int main() {
    // << is the insertion operator
    std::cout << "Hello, Forge!" << std::endl;
    return 0;
}

Namespaces

To avoid naming collisions (e.g., your Physics class vs. a library's Physics class), we wrap code in namespaces.

cpp
namespace Engine {
    int id = 1;
}

int main() {
    // Access via Scope Resolution Operator (::)
    std::cout << Engine::id;
}

References

A safer, cleaner alternative to pointers. A reference is an alias for an existing variable. It must be initialized and cannot be null.

cpp
int hp = 100;
int &ref = hp; // ref is now an alias for hp

ref -= 10; // hp is now 90

II. Apprentice (The STL)

The Standard Template Library (STL) provides optimized data structures. Never write a linked list from scratch unless you are studying.

Vectors

Dynamic arrays that handle memory management automatically.

cpp
#include <vector>

std::vector<int> nums = {1, 2, 3};
nums.push_back(4); // Adds to end

// Range-based For Loop (Modern C++)
for (int n : nums) {
    std::cout << n << " ";
}

Strings

The std::string class manages character arrays for you.

cpp
#include <string>

std::string s1 = "Aether";
std::string s2 = "Syntax";

// Concatenation
std::string full = s1 + " " + s2; 

// Comparison (Works unlike C's strcmp)
if (s1 == "Aether") { /* ... */ }

Maps

Key-value pairs (usually implemented as Red-Black Trees).

cpp
#include <map>

std::map<std::string, int> inventory;
inventory["Potion"] = 5;
inventory["Sword"] = 1;

III. Journeyman (OOP)

Classes allow us to bundle data and logic, hiding the complexity (Encapsulation).

Classes

By default, class members are private.

cpp
class Player {
private:
    int health; // Internal state

public:
    // Constructor (Initializer List is preferred)
    Player(int h) : health(h) {}

    void takeDamage(int dmg) {
        health -= dmg;
    }
};

Player p1(100);
p1.takeDamage(10);

Inheritance & Polymorphism

Allowing classes to share behavior. Use virtual functions to allow overriding.

cpp
class Entity {
public:
    virtual void draw() { std::cout << "Drawing Entity"; }
};

class Enemy : public Entity {
public:
    // Override ensures signature matches base
    void draw() override { std::cout << "Drawing Enemy"; }
};

IV. Expert (Modern Memory)

Rule: Avoid new and delete. Modern C++ uses RAII (Resource Acquisition Is Initialization) and Smart Pointers to prevent memory leaks automatically.

Unique Pointer (std::unique_ptr)

Exclusive ownership. When the pointer goes out of scope, the memory is freed. Cannot be copied, only moved.

cpp
#include <memory>

// Create a unique pointer to a Player
auto ptr = std::make_unique<Player>(100);

// ptr->takeDamage(5);
// Memory freed automatically here

Shared Pointer (std::shared_ptr)

Shared ownership via reference counting. Memory is freed only when the last pointer to the object is destroyed.

cpp
auto s1 = std::make_shared<Player>(100);
auto s2 = s1; // Both point to the same object. Ref count = 2.

Lambdas

Anonymous functions. Useful for callbacks and passing logic to STL algorithms.

cpp
int multiplier = 2;

// [capture](args) { body }
auto times = [multiplier](int a) {
    return a * multiplier;
};

std::cout << times(5); // 10

V. Grandmaster (Metaprogramming)

Writing code that generates code.

Templates

Generic programming. Write a function once, work with any type.

cpp
template <typename T>
T add(T a, T b) {
    return a + b;
}

int x = add<int>(5, 10);
float y = add<float>(3.5, 4.5);

Move Semantics (std::move)

Performance optimization. Instead of copying heavy data (deep copy), we "steal" the pointers (shallow copy) from temporary objects.

cpp
std::string a = "Heavy Data";
// 'b' takes ownership of a's buffer. 'a' is left empty.
std::string b = std::move(a);

Concurrency

Multi-threading with the standard library.

cpp
#include <thread>

void task(int id) { /* work */ }

std::thread t1(task, 1);
t1.join(); // Wait for thread to finish
