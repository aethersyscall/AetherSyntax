C Programming

The Mother of Modern Languages.

C provides low-level access to memory, a small keyword set, and a clean style. It is the language of operating systems, embedded architectures, and high-performance computing. It trusts the programmer implicitlyâ€”for better or worse.

I. Novice (The Foundation)

C is a compiled language. Every program starts execution at the main function.

The Structure

Header files are required to access standard library functions like printing.

c
// Include standard Input/Output library
#include <stdio.h>

int main() {
    printf("Hello, Forge!\n");
    
    // Return 0 indicates success to the OS
    return 0; 
}

Variables & Types

C is statically typed. Types dictate how much memory is allocated.

c
int id = 42;          // Integer (usually 4 bytes)
float pi = 3.14;      // Floating point
double precise = 3.14159; // Double precision
char grade = 'A';     // Single character (1 byte)

// Constants
const int MAX_USERS = 100;

Input & Output

Use format specifiers to match the variable type.

  • %d: Integer
  • %f: Float
  • %c: Char
  • %s: String
c
int age = 25;
printf("User is %d years old.\n", age);

// Input (Pass the address using &)
scanf("%d", &age);

II. Apprentice (Control & Arrays)

Handling logic flow and collections of data.

Control Flow

Standard logic structures. Braces {} are mandatory for multi-line blocks.

c
if (score > 90) {
    printf("Rank: S");
} else if (score > 75) {
    printf("Rank: A");
} else {
    printf("Rank: B");
}

// Loops
for (int i = 0; i < 5; i++) {
    printf("%d ", i);
}

while (is_running) {
    // Game loop logic
}

Arrays

Fixed-size collections. Warning: C does not check array bounds. Accessing index 10 of a size 5 array will crash or corrupt memory.

c
int numbers[5] = {10, 20, 30, 40, 50};
printf("%d", numbers[0]); // 10

Strings

C has no string type. Strings are just arrays of characters ending with a Null Terminator (\0).

c
char name[] = "Aether";
// In memory: ['A', 'e', 't', 'h', 'e', 'r', '\0']

III. Journeyman (Pointers)

The barrier to entry. A pointer is a variable that stores a memory address.

The Syntax

  • &: Address-of operator (Get the address).
  • *: Dereference operator (Go to the address and get the value).
c
int val = 10;
int *ptr = &val; // ptr holds the address of val

printf("%p", ptr);  // Prints memory address (e.g., 0x7ffee...)
printf("%d", *ptr); // Prints 10 (Value at that address)

*ptr = 20; // val is now 20

Pointer Arithmetic

Arrays decay into pointers. Moving a pointer moves it by the size of the data type.

c
int arr[] = {10, 20, 30};
int *p = arr; // Points to arr[0]

printf("%d", *(p + 1)); // Prints 20 (arr[1])

IV. Expert (Memory & Structs)

Building complex data types and managing the Heap.

Structs

Grouping related data. The precursor to Objects.

c
struct Player {
    char name[50];
    int hp;
    int score;
};

// Initialization
struct Player p1 = {"Shadow", 100, 0};

// Accessing fields
printf("%s", p1.name);

// Pointers to Structs (Arrow Operator)
struct Player *ptr = &p1;
ptr->hp -= 10; // Same as (*ptr).hp

Dynamic Memory (The Heap)

Stack memory is automatic. Heap memory must be manually requested (malloc) and released (free).

c
#include <stdlib.h>

// Allocate memory for 10 integers
int *arr = (int*)malloc(10 * sizeof(int));

if (arr != NULL) {
    arr[0] = 5;
}

// CRITICAL: You must free memory to prevent leaks
free(arr);
arr = NULL; // Good practice to avoid dangling pointers

V. Grandmaster (Systems Programming)

Code that talks directly to hardware and the compiler.

Preprocessor Macros

Directives that run before compilation.

c
#define PI 3.14159
#define SQUARE(x) ((x) * (x))

// Conditional Compilation
#ifdef DEBUG
    printf("Debug Mode ON");
#endif

Bitwise Operations

Manipulating individual bits. Essential for embedded systems and drivers.

  • &: AND
  • |: OR
  • ^: XOR
  • <<: Left Shift
  • >>: Right Shift
c
unsigned char flags = 0x01; // 0000 0001

// Set the 3rd bit
flags |= (1 << 2); // Result: 0000 0101

// Check if 3rd bit is set
if (flags & (1 << 2)) {
    // True
}

Function Pointers

Storing a function in a variable to pass logic around (Callbacks).

c
void execute(void (*func)(int)) {
    func(42);
}

void print_val(int x) {
    printf("Val: %d", x);
}

int main() {
    execute(print_val); // Passes function as argument
}
