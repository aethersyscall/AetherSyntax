JavaScript: Logic

The Behavior of the Web.

HTML builds the house. CSS paints it. JavaScript makes the lights turn on, the doors open, and the elevators move. It is an event-driven, dynamic language.

I. Novice (The Basics)

Forget var. Modern JavaScript relies on strict variable declarations to prevent bugs.

Variables

  • const: The default. Use this if the value won't be reassigned.
  • let: Use this only if the value must change (e.g., counters).
javascript
const appName = "AetherSyntax"; // Cannot be changed
let userCount = 0;              // Can change

userCount++; // 1

The Console

Your primary tool for debugging.

javascript
console.log("System initialized.");
console.error("Connection failed.");

// Displays data as a clean table
console.table([
  { id: 1, status: "Active" },
  { id: 2, status: "Offline" }
]);

Data Types

JavaScript is loosely typed (you don't declare types), but types still exist.

javascript
const name = "Forge";       // String
const score = 98.5;         // Number
const isActive = true;      // Boolean
const empty = null;         // Explicitly empty
const unknown = undefined;  // Implicitly empty

II. Apprentice (Modern Syntax)

ES6 (2015) revolutionized JavaScript. Writing "Old JS" is now considered a bad practice.

Template Literals

Stop using + to join strings. Use backticks ` and ${}.

javascript
const user = "Shadow";
const level = 5;

// The Old Way (Bad)
// const msg = "User " + user + " is level " + level;

// The Modern Way (Good)
const msg = `User ${user} is level ${level}`;

Arrow Functions

A concise syntax for writing functions.

javascript
// Traditional Function
function add(a, b) {
  return a + b;
}

// Arrow Function (Implicit Return)
const add = (a, b) => a + b;

// Block Body (Requires 'return')
const process = (data) => {
  const result = data * 2;
  return result;
};

III. Journeyman (The DOM)

The Document Object Model is how JS talks to HTML.

Selection (querySelector)

The Swiss Army Knife of selecting elements. It uses CSS selectors.

javascript
// Select by ID
const header = document.querySelector("#main-header");

// Select by Class (Returns first match)
const card = document.querySelector(".card");

// Select All (Returns a NodeList/Array)
const allButtons = document.querySelectorAll("button");

Event Listeners

Waiting for the user to do something.

javascript
const btn = document.querySelector("#submit-btn");

btn.addEventListener("click", (event) => {
  event.preventDefault(); // Stop form from reloading page
  console.log("Button clicked!");
  
  // Modify styles directly
  btn.style.backgroundColor = "#9580FF";
});

IV. Expert (Data Manipulation)

Handling complex data structures efficiently is the hallmark of a senior engineer.

Destructuring

Unpacking values from objects or arrays into distinct variables.

javascript
const config = {
  theme: "Eclipse",
  version: "2.0",
  debug: true
};

// Extract 'theme' and 'version' into variables
const { theme, version } = config;

console.log(theme); // "Eclipse"

The Spread Operator (...)

Copying or merging data without mutating the original source.

javascript
const baseStats = { hp: 100, mp: 50 };
const buffedStats = { ...baseStats, hp: 150, armor: 20 };

// Result: { hp: 150, mp: 50, armor: 20 }

const nums = [1, 2, 3];
const allNums = [...nums, 4, 5, 6];

Array Methods

Loops (for) are verbose. Use functional methods instead.

  • map: Transform every item.
  • filter: Select specific items.
javascript
const prices = [10, 20, 30, 40];

// Double every price
const doubled = prices.map(price => price * 2);
// [20, 40, 60, 80]

// Keep only prices over 25
const expensive = prices.filter(price => price > 25);
// [30, 40]

V. Grandmaster (Async & Network)

JavaScript is single-threaded, but it handles long tasks (like fetching data) using the Event Loop.

Fetch API

Getting data from a server.

javascript
fetch("https://api.aether.com/users")
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error("Network Error:", error));

Async / Await

Syntactic sugar over Promises. It makes asynchronous code look synchronous and readable.

javascript
async function loadUserData() {
  try {
    const response = await fetch("https://api.aether.com/users");
    
    if (!response.ok) throw new Error("Server Error");
    
    const data = await response.json();
    return data;
    
  } catch (error) {
    console.error("Failed to load:", error);
  }
}

// Usage
loadUserData();

Modules

Splitting code into multiple files.

javascript
// math.js
export const add = (a, b) => a + b;

// main.js
import { add } from './math.js';
console.log(add(2, 2));
